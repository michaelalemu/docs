<Update label="February 2025">
  ## Changelog

  Introducing an embeddable exchange rate widget. Easily display currency rates on your website with a customizable and lightweight widget.

  <iframe height="300px" width="100%" src="https://fxethio.com/widget/exchange-rates?background=lightgrey&currencies=USD,EUR,GBP,CNY" />

  ### Features

  * **Embeddable Widget**: Easily embed exchange rate widgets on any website.

  * **Customizable UI**: Supports different themes (`light` & `dark`) and background colors.

  * **Multi-Currency Support**: Fetch rates for USD, EUR, GBP, JPY, and more.

  * **Responsive Design**: Works on all screen sizes, including mobile.

  * **Optimized Performance**: Lightweight and efficient with cached exchange rates.

  ### How to Use

  1. Copy the embed code from the home page.

  2. Paste it into any HTML page.

  3. Customize with URL parameters like `?background=lightgrey&currencies=USD,EUR`. Check the API Documentation for all options.
